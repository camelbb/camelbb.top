1.安装步骤

package 名称

go build 编译代码

go clean 清除编译生成的代码

go fmt 格式化代码

go get 获取远程代码包

go install
这个命令在内部实际上分成了两步操作：第一步是生成结果文件(可执行文件或者.a包)，第二步会把编译好的结果移到$GOPATH/pkg或者$GOPATH/bin。

go test
会自动读取源码目录下面名为*_test.go的文件，生成并运行测试用的可执行文件

godoc
文档工具，很强悍
godoc -http=:端口号 生成web版文档，通过http访问

go version 查看go当前的版本
go env 查看当前go的环境变量
go list 列出当前全部安装的package
go run 编译并运行Go程序
