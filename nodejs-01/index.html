npm模块安装路径

==============================================================

npm安装package.json中的模块依赖

####　1.package.json不存在时
命令:npm init可自动创建package.json文件

2.package.json存在时

直接命令:npm install 或者 npm install –save-dev会自动将package.json中的模块安装到node-modules文件夹下

================================================================
使用淘宝镜像

npm install cnpm -g

卸载包
npm uninstall xxx
npm uninstall xxx -g

列出全域套件
npm ls -g

列出全域套件詳細資訊.
npm ls -gl

更新全域套件.
npm update -g
